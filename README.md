# PassCode Creator – Strong Password Generator

PassCode is a simple password generator designed to create strong and secure passwords directly on your device.

## About the App

PassCode allows you to generate secure passwords with customizable options including:

• Adjustable password length  
• Symbols, numbers, and uppercase letters  
• Word-based password generation  
• Password strength indicator  

All passwords are generated locally on your device.

No passwords are stored or transmitted.

## Privacy

PassCode does not collect, store, or share any personal data.
