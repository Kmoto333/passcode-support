## About the App

PassCode Creator allows you to generate and manage secure passwords 
with features including:

- Adjustable password length
- Symbols, numbers, uppercase and lowercase letters
- Optional word-based password generation
- Password strength indicator with 5 levels
- Password history (last 10 generated passwords)
- Save up to 5 passwords locally (up to 50 with Premium)
- Secure access to saved passwords via device passcode or Face ID / Touch ID
- Organise saved passwords by category and sort options
- Share passwords via system share options
- 9 themes including 6 premium visual themes

All passwords are generated locally on your device.  
Saved passwords and history are stored only on your device 
and are never transmitted.
